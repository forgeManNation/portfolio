Projekty

Mapa mezivládních organizací

Interaktivní mapa, která ukazuje mezivládní organizace. Data vznikla pomocí dat z World Fact Book. JSON soubor, obsahující data organizací, byl vytvořen s pythonem, metodou data scraping. Aplikace je vytvořena v Reactu a v Typescriptu za využití grafické knihovny Tailwind a mapa potom pomocí React-Leaflet.

Typescript React.js Tailwind Leaflet Python

   

Analýza kompetitivních hypotéz

Kdo zavraždil Kennedyho? Je ve fotbalu důležitější silná obrana nebo útok? Když nemáte numerická data, ale spoustu důkazů a hypotéz, použijte tuto aplikaci.
Po vyplnění analýzy se vám ukáže výsledná pravděpodobnost v procentech. Aplikace vytvořena v Reactu. Grafické rozhraní je kombinací Bootstrap 5 a Sass. Tato aplikace využívá user-side i server-side databázi, Redux a Firebase.

Javascript React.js Redux Bootstrap Firebase (Firestore, Authentification) Sass Leaflet

   

Event Miner

Čistě frontend aplikace pro poznávání časových, geografických a ostatních údajů v textu. Aplikace vytvořená při stáži u jedné nadnárodní korporace, kvůli ochraně dat je k dispozici pouze případová studie. Aplikace je psána v Reactu a s využitím Bootstrap 4.

Javascript React.js Bootstrap

Deník

Online deník vytvořený pro cestovatele. Uživatel může přidávat obráyky mapy a libovolné množství stránek. Vytvořen v React.js, dále využito firebase cloud storage, firestore a fireauth, grafika v Scss, routování pomocí React Routeru.

Typescript React.js Sass React-Router Firebase (Firestore, Authentification, Cloud Storage)
